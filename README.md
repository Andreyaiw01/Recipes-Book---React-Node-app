Recipes Book builded with React + Node.js <br />
You can add / rewrite recipes, and see change history of recipes. <br />
Recipes have been sorted by date.

## Start Application

In the project directory, you can run:

### `npm start`

Runs the app (server side + client side) in the development mode.<br />
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

The page will reload if you make edits.<br />
You will also see any lint errors in the console.
